# Stock Market Prediction (Python + Tkinter)

This is an educational machine learning project that demonstrates
how historical stock data can be analyzed using Python.

## Features
- Data preprocessing and feature engineering
- Linear Regression and Random Forest models
- Model comparison using MAE
- Desktop GUI built with Tkinter
- Data visualization with Matplotlib

## Project Status
This project focuses on learning, structure, and design.
Accuracy and performance are not the primary goals.
Sample dataset provided for testing.
